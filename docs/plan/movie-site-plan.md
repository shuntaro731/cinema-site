# 映画紹介サイト 実装プラン

## 目的

映画館で上映する映画を、ブラウン管のような湾曲した画面表現で紹介するサイトを作る。映画情報はプロジェクト内の静的データで管理し、背景には各映画のティザー動画を使う。

## 実装方針

- Astroで静的寄りのサイトとして構築する
- スタイルはTailwindを中心に記述する
- 映画ごとに1画面のセクションを作り、縦スクロールで次の映画へ移動する
- ブラウン管の `imageViewer` をクリックすると画面いっぱいに拡大し、映画詳細ページへ遷移する
- タイトル、概要、公開日、コピー、ティザー動画URLを `src/data/movies.ts` で管理する
- ティザー動画はローカル配置したファイルを映画データに紐づけて表示する

## 作業手順

1. Tailwindを導入し、Astroの初期画面を映画紹介サイト用に置き換える
2. 映画データの型を定義し、タイトル、概要、上映日、ティザー動画URLをまとめて扱える形にする
3. ローカル動画を読み込めるように映画データへ動画ソースを登録する
4. `MovieScreen` コンポーネントを作り、映画1本分の表示を分離する
5. 背景にティザー動画を全面表示し、暗いオーバーレイで文字の可読性を確保する
6. 画面全体にCRT風の湾曲、黒い外枠、スキャンライン、ノイズ表現を重ねる
7. 縦スクロールに `snap-y` を使い、1スクロールで次の映画へ移動しやすくする
8. `imageViewer` クリック時にCRT画面がビューポートいっぱいへ広がる演出を入れる
9. 拡大演出の完了後、`/movies/[id]` の映画詳細ページへ遷移する
10. 詳細ページで静的データのタイトル、概要、公開日、ティザー動画などを表示する
11. 動画がない映画では暗いグラデーション背景を代替表示する
12. 動画なし、概要なしの表示を用意する
13. `bun run build` とブラウザ表示でレイアウト崩れを確認する

## 想定ファイル

- `src/data/movies.ts`: 表示対象の映画情報とティザー動画URL
- `src/components/MovieScreen.astro`: 映画1本分の画面
- `src/components/CrtFrame.astro`: ブラウン管風フレーム
- `src/components/ImageViewer.astro`: クリック可能なCRT画面領域
- `src/pages/index.astro`: スクロール全体のページ
- `src/pages/movies/[id].astro`: 映画詳細ページ
- `public/videos/`: ティザー動画をローカル配置する場合の置き場

## 注意点

- 映画ごとの動画URLは手動で紐づける
- 動画は自動再生のため `muted`、`playsinline`、`loop` を指定する
- クリックから遷移までの拡大演出は短くし、操作の重さを感じさせない
- 拡大演出中はスクロール操作を止め、二重クリックで遷移が重複しないようにする
- 文字情報は動画の上に置くため、暗いグラデーションまたは黒の半透明レイヤーを必ず入れる
- ピクセルパーフェクトよりも、映画ごとの画面サイズ、余白、テキスト位置の規則性を優先する
