# CRT映画紹介サイト デザイン仕様

## コンセプト

映画のティザー映像を、古いブラウン管モニターに映しているように見せる。画面は暗く、映像は少し歪み、黄色がかった文字で映画館らしい重さと懐かしさを出す。

## 画面構成

- 1つの映画を1画面として表示する
- 背景は映画ごとのティザー動画を全面に敷く
- 中央付近に映画タイトル、公開日、短いコピーを配置する
- 下部に大きな日付または上映開始日を表示する
- スクロールすると次の映画の画面へ移動する
- ブラウン管の `imageViewer` をクリックすると画面がビューポートいっぱいに広がり、その映画の詳細ページへ遷移する

## CRT表現

- 画面の四隅を大きく湾曲させる
- 外側は黒い余白を広めに取り、モニターの縁のように見せる
- 動画の上に薄いスキャンラインを重ねる
- 全体に粒子感のあるノイズを重ねる
- 画面中央は少し明るく、端に向かって暗くする
- 完全な長方形ではなく、少し内側へ沈むような奥行きを出す

## 配色

- 背景: 黒、焦げ茶、暗いグレー
- メイン文字: 黄味のある白、淡いクリーム
- 強調文字: 古い蛍光表示のような黄色
- 補助線: 低彩度の黄色または薄いブラウン

## タイポグラフィ

- タイトルは映画館のポスターや古い字幕を連想するセリフ系を優先する
- 数字や日付は太く、大きく、画面下部に強く出す
- 説明文は短くし、長文を載せすぎない
- 文字間は詰めすぎず、中央揃えを基本にする

## レイアウトルール

- 各映画セクションの情報配置は揃える
- 中央のタイトルブロックと下部の日付の縦位置を統一する
- 画面端ぎりぎりに文字を置かない
- `imageViewer` はクリックできる主役の領域として扱い、ホバー時にわずかな明るさや枠線の変化を付ける
- Pencilの比率よりも、行・列・余白の揃いを優先する
- 過度なレスポンシブ調整は入れず、基本の縦積みと中央配置で成立させる

## 詳細ページ遷移

- トップ画面ではブラウン管の表示領域全体をクリック対象にする
- クリック時はCRTの丸みと外枠を保ったまま、短い時間で画面いっぱいに拡大する
- 拡大が完了したら `/movies/[id]` に遷移する
- 詳細ページはトップと同じ映画のティザー動画または背景画像を使い、映画情報をより詳しく見せる
- 戻ったときにトップのスクロール体験が破綻しないよう、演出はトップページ内で完結させる

## 動画仕様

- 動画は `video` タグで背景表示する
- `autoplay muted loop playsinline` を指定する
- 画面いっぱいに `object-cover` で表示する
- 動画が読み込めない場合はTMDBの背景画像を表示する
- 音声再生は初期実装では扱わない

## Tailwind実装メモ

- セクション全体: `relative h-screen snap-start overflow-hidden bg-black`
- 背景動画: `absolute inset-0 h-full w-full object-cover`
- 暗い重ね: `absolute inset-0 bg-black/50`
- 中央情報: `relative z-10 flex h-full flex-col items-center justify-center text-center`
- クリック領域: `group relative block cursor-pointer overflow-hidden`
- 拡大中: `fixed inset-0 z-50 h-screen w-screen`
- CRT外枠: `rounded-[10%]` など必要最小限の任意値を使う
- スキャンラインやノイズは疑似要素または薄いオーバーレイで実装する

## 優先する見た目

1. 動画が映画ごとに切り替わること
2. 四隅が湾曲したブラウン管らしさが出ていること
3. スクロールで次の映画へ自然に移動できること
4. `imageViewer` クリック時に画面いっぱいへ広がる遷移が気持ちよくつながること
5. 映画タイトルと上映日が読みやすいこと
6. 余白とテキスト位置に規則性があること
